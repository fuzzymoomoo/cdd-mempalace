# Contributing To cdd-mempalace

Thanks for taking a look.

This project exists to make the connection between CDD and MemPalace practical, understandable, and useful in public.

## Good Contributions

- clearer mapping docs
- better examples
- retrieval evaluation notes
- public-safe engineering scenarios
- upstream contribution ideas backed by tested findings

## Ground Rules

- do not imply official endorsement from MemPalace
- keep the relationship to upstream explicit and respectful
- prefer tested findings over vague claims
- document tradeoffs and failures, not only successes
- keep examples understandable without private context

## Before Large Changes

If a change alters the core mapping or public positioning, open an issue or discussion first so the repo stays coherent.
